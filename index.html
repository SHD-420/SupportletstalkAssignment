Assignment API Overview

  • GET

    /data/

    List all the document ids.

  • GET

    /data/:id

    Get a detailed document with :id (mongodb _id)

  • POST

    /data/

    Write a document.

  • PATCH

    /data/:id

    Update a document with :id (mongodb _id)

  • DELETE

    /data/:id

    Delete a document with :id (mongodb _id)
